# Definition of Done (DoD)

> Definida formalmente no Scrum Guide (Schwaber e Sutherland) como o compromisso associado ao artefato Incremento.

## O que é

Uma descrição formal do estado em que o Incremento deve estar para atender às medidas de qualidade exigidas do produto. No momento em que um item do Product Backlog atende à Definition of Done, um Incremento nasce.

## Papel no Scrum

- Cria **transparência** ao fornecer entendimento compartilhado sobre o que significa "o trabalho está completo".
- Se a Definition of Done para um Incremento faz parte dos padrões da organização, todos os Scrum Teams devem segui-la no mínimo; se não é uma norma organizacional, o Scrum Team deve criar uma Definition of Done apropriada para o produto.
- Se um item do Product Backlog não atende à Definition of Done, ele **não pode ser lançado nem apresentado** na Sprint Review — retorna ao Product Backlog para consideração futura.

## Exemplos típicos de critérios (variam por time/organização)

- Código revisado (code review) e integrado à branch principal.
- Testes automatizados escritos e passando (unitários, integração).
- Critérios de aceitação da história validados.
- Documentação atualizada, quando aplicável.
- Sem defeitos críticos/bloqueadores conhecidos.
- Aprovado pelo Product Owner.

*(A lista concreta de critérios é específica de cada organização/produto — a norma não prescreve os itens exatos, apenas exige que existam e sejam usados de forma consistente.)*

## Diferença em relação à DoR

A DoD avalia a **saída** do trabalho (quando um item pode ser considerado terminado); a DoR avalia a **entrada** (quando um item está maduro o bastante para começar) — ver [[dor]]. São complementares, não intercambiáveis.

## Relevância para este agente

A DoD é o padrão de qualidade contra o qual os critérios de aceitação gerados por este agente devem eventualmente ser confrontados: um critério de aceitação bem escrito e testável (ver [[invest]], [[iso29148]]) é o que permite verificar objetivamente se a DoD foi atingida para aquele item.
